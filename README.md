# Python-StudentAverageMark-sCalculatorUsingOOP
This is a simple Python mini project that calculates the average marks of a student using Object-Oriented Programming (OOP) concepts.  
<br>
<b>Features</b>: - Takes input for 5 subjects - Uses class and object implementation - Calculates average marks - Displays personalized welcome message.
<br>
<br>
This project is created for practice and learning basic Python concepts like classes, loops, and user input handling.
